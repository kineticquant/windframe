Windframe - Tailwind CSS editor



How To Use Windframe

Simply drag and drop elements from the components list on the left into the canvas. When finished with your design, export the canvas as HTML via the Export Design button at the bottom of the page. This will download a windframe.html file which can be further modified to your liking.

About

This utility was developed out of necessity to rapidly add Tailwind elements to a webpage and having no desire to pay for any premium services when doing so. It's not an entire website builder, but instead a rapid proof-of-concept designer utility that can be quickly exported and added into ongoing development projects.

Please note this utility is still under active development.
